鼠标点击烟花爆炸效果

作者链接：https://www.iowen.cn/canvas-click-effect-second-edition/

js下载：https://wwa.lanzous.com/idow0oyhoqf

打开 themes-主题名-config.yml 文件，找到 

custom_js:

引入js

- //js文件链接 #烟花效果

在 themes-主题名-layout-layout.ejs 文件 body 标签中加入 

<div id="clickCanvas" style=" position:fixed;left:0;top:0;z-index:999999999;pointer-events:none;"></div>

首页添加音乐

下载：https://cdnjs.com/libraries/aplayer

新建dist文件夹，把下载下来的文件全放进去

在dist目录里，新建 music.js 文件，并把如下代码粘贴进去。

const ap = new APlayer({
    container: document.getElementById('aplayer'),
    fixed: true,
    autoplay: true,
    audio: [
      {
        name: "歌曲名",
        artist: "歌手",
        url: '音频链接',
        cover: '图片链接',
      }
    ]
});

全放到source文件夹里

在themes-主题名-layout-layout.ejs文件中，body 标签里新增如下代码

<!-- 音频播放 -->
<link rel="stylesheet" href="/dist/APlayer.min.css">
<div id="aplayer"></div>
<script type="text/javascript" src="/dist/APlayer.min.js"></script>
<script type="text/javascript" src="/dist/music.js"></script>

背景添加动态线条效果

js下载：https://wwa.lanzous.com/iz677oyhosh

在 themes-主题名-layout-layout.ejs 文件中添加如下代码：

<!--动态线条背景-->
<script type="text/javascript"
color="145,128,114" opacity='0.7' zIndex="-2" count="200" src="js文件链接">
</script>

其中：
color：表示线条颜色，三个数字分别为(R,G,B)，默认：（0,0,0）
opacity：表示线条透明度（0~1），默认：0.5 - count：表示线条的总数量，默认：150
zIndex：表示背景的z-index属性，css属性用于控制所在层的位置，默认：-1

添加console.log字符画

把以下代码放到 themes-主题名-layout-layout.ejs 文件 body 下面

<!-- console.log -->
<script>
    if (window.console) {
        console.log("你要显示的文字");
        console.log("你要显示的文字");
    }
</script>
<script>
    function makeMulti (string) {
        let l = new String(string)
        l = l.substring(l.indexOf("/*") + 3, l.lastIndexOf("*/"))
        return l
    }
    let string = function () {
      /* 你要显示的字符画放到这（替换这些文字） */
    }
    console.log(makeMulti(string));
</script>

字符画可以用Ascgen2转换，软件自行百度

修改博客字体

选择自己喜欢的字体打开这个网站进行转换

打开source文件夹，新建一个font文件夹

转换出来的东西都放进去

打开转换生成的css文件，url改为字体文件链接

@charset "UTF-8";

@font-face {
  font-family: "Muyao-Softbrush";
  src: url("/font/xxxxx.woff2") format("woff2"),
       url("/font/xxxxx.woff") format("woff"),
       url("/font/xxxxx.ttf") format("truetype"),
       url("/font/xxxxx.eot") format("embedded-opentype"),
       url("/font/xxxxx.svg") format("svg");
  font-weight: normal;
  font-style: normal;
}

打开 themes-主题名-config.yml 文件，找到

custom_css:

引入css

- //xxxxx.css的链接

或者说也可以在 themes-主题名-source-css-main.styl 中输入

@import "xxxxx.css的链接"

打开主题配置文件config.yml，找到

font_family:

改成你的字体

如果主题配置文件中找不到font_family:可以把 themes-主题名-source-css-variables.style 文件中font-family 修改成你的字体就可以了(没有试过，理论上可行)

Fluid设置背景图片

首先我们得把主题中首屏图片关掉，不然直接设置背景不好看

打开themes-fluid-_config.yml

首屏图片的链接删掉，然后把透明度改成0

这样首屏我们就看不到了

然后打开 themes-fluid-layout-layout.ejs ，最上面输入

<style type="text/css">
body {
	background-image: url(你的背景图片链接);
	background-attachment: fixed; //让内容滚动的时候背景图片不动
    background-size:cover; //背景图片铺满整个页面 图片比例保持不变
}
</style>

然后往下拉，找到

<%- partial('_partial/nav') %>
  <div class="banner" id="banner" <%- theme.banner && theme.banner.parallax && 'parallax=true' %>
       style="background: url('<%- url_for(banner_img) %>') no-repeat center center;
         background-size: cover;">

url里面的给删掉

background: url ('')

因为主题配置文件里的banner链接我们已经删掉了，所以这个我们也给删掉，不然控制台里会有报错

GET http://xxxxx/null 404 (Not Found)

这样就好了

    });
